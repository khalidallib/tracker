# GoRide - Driver tracking map

## Features
- Shows your location on a Mapbox map
- Updates in real time as you move
- Marker highlights your position
- Track button to center the map on you

## Setup
1. Get a free Mapbox account and access token from [mapbox.com](https://www.mapbox.com/).
2. Replace the token in `script.js` with your own.
3. Open `index.html` in your browser.
